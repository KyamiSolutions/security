🌐 Hosting
Loo automaatselt veebimajutus oma domeenile — Cloudflare DNS, tunnel ja Virtualmini virtuaalserver
📋 Kuidas see töötab
1
Lisa domeen Cloudflare'i (ühekordne)
Mine Cloudflare → Add Site → sisesta domeen (nt kalle.ee) → vali Free plaan. Cloudflare annab kaks nameserverit. Mine oma registrari (Elkdata, Namecheap jne) → asenda Nameservers nendega. Propageerub 5 min — 24 h.
2
Sisesta domeen all olevasse vormi
Kirjuta täisdomeen — blogi.mrnux.ee, kalle.ee või www.kalle.ee — ja vajuta Loo. Süsteem teeb taustal: leiab Cloudflare tsooni, loob DNS CNAME, lisab tunneli routele, käivitab Virtualmini virtuaalserveri.
3
Halda saiti Virtualminis
Kuvatakse loodud Virtualmini parool — kirjuta turvaliselt üles. Logi sisse aadressil panel.mrnux.ee ja halda faile, andmebaase, e-maile.
➕ Lisa uus domeen
💡 Näide: kui registreerisid kalle.ee ja muutsid nameservers Cloudflare'is, saad siia kirjutada nii kalle.ee kui ka www.kalle.ee või suvalise alamdomeeni.

❓ Korduvad küsimused
Mis juhtub kui sisestan domeeni, mille tsoon pole minu Cloudflare kontol?
Saad veateate "Cloudflare tsooni ei leitud". Süsteem ei saa programmiselt teiste kontode tsoone hallata. Lahendus: lisa domeen oma Cloudflare kontosse (Add Site) ja muuda registrari nameservers.
Saan ma hostida kellegi teise domeeni?
Jah — domeeni omanik ei pea Cloudflare kontot omama. Sina lisad domeeni oma Cloudflare'i, omanik vaid muudab oma registraris nameservers Cloudflare omadeks.
Kuidas SSL töötab?
SSL'i pakub Cloudflare automaatselt (Flexible SSL). Brauser → Cloudflare HTTPS, Cloudflare → tunnel HTTP. Otsast-otsa TLS-i jaoks: Virtualmini all Let's Encrypt sertifikaat.
Mis juhtub kui kustutan domeeni siit?
Kõik korraga: DNS kirje (Cloudflare), tunnel route, Virtualmini virtuaalserver + kõik failid ja andmebaasid. Tegevus on pöördumatu.
Kuidas vaadata kas DNS on propageerunud?
Käsk: dig kalle.ee peab tagastama Cloudflare IP'd (104.21.x.x või 172.67.x.x). Kui mitte — registraris pole nameservers veel Cloudflare omadeks vahetatud.
